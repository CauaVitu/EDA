## 🚀 Visão Geral do Projeto

O objetivo deste repositório é consolidar o conhecimento prático sobre como organizar e manipular dados de forma otimizada. Cada estrutura é implementada focando em clareza lógica e performance.

## 🏗️ Conteúdo do Repositório

Abaixo estão as estruturas de dados e algoritmos organizados por categoria:

### 1. Estruturas Lineares
* **Listas Encadeadas:** Simples, Duplas e Circulares.
* **Pilhas (Stacks):** Lógica LIFO (Last-In, First-Out).
* **Filas (Queues):** Lógica FIFO (First-In, First-Out).

### 2. Estruturas Não-Lineares (Hierárquicas)
* **Árvores Binárias:** Árvores de Busca Binária (BST).
* **Árvores Balanceadas:** AVL e Rubro-Negra (Red-Black).
* **Heaps:** Filas de prioridade com Min-Heap e Max-Heap.

### 3. Outras Estruturas
* **Tabelas Hash:** Implementações com diferentes tratamentos de colisão.
* **Grafos:** Representação por matriz e lista de adjacência.



---

## 📈 Análise de Complexidade (Big O)

Cada algoritmo busca atingir a melhor eficiência teórica:

| Estrutura | Busca | Inserção | Remoção |
| :--- | :---: | :---: | :---: |
| **Lista Encadeada** | $O(n)$ | $O(1)$ | $O(1)$ |
| **Árvore Binária (BST)** | $O(log\ n)$ | $O(log\ n)$ | $O(log\ n)$ |
| **Tabela Hash** | $O(1)$ | $O(1)$ | $O(1)$ |
